# Meme Generator (React)

A simple React-based meme generator that allows users to create custom memes by adding text over randomly fetched meme images.

## Features
- Fetches meme images from the Imgflip API
- Generates a random meme image on button click
- Allows users to add:
  - Top text
  - Middle text
  - Bottom text
- Real-time text updates on the meme
- Simple and responsive UI

## Tech Stack
- React (Functional Components + Hooks)
- JavaScript (ES6+)
- CSS
- Fetch API

## How It Works
1. The app fetches meme templates from the Imgflip API.
2. A random meme image is selected.
3. User enters text in the input fields.
4. Text is displayed over the meme image in real time.
